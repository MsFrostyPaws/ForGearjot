# Gearjot

API for checking prime numbers.

You can simply run it by:\
`
make run
`

And simply test by \:

`
make test
`

### Added:
- Benchmarks to tests;
- Thoughts on the future of this app;
- A rest client (tests/rest);
- Gin - a HTTP web framework. It is Fast, Crash-free and Extendable;
